# Supply-chain
This Power BI dashboard is about supply chain and logistics operations, helping company to monitor order fulfillment, delivery performance, inventory flow, and shipping efficiency.

1. Project Title / Headline
Supply Chain & Logistics Analytics Dashboard (Power BI)


2. Short Description about project
This Power BI dashboard analyzes supply chain and logistics data to track order processing, shipment performance, delivery timelines, and inventory movement.
It is designed to help businesses identify inefficiencies, improve delivery reliability, and optimize overall supply chain operations through interactive visual insights.


3. Tech Stack
Power BI Desktop – Data modeling, DAX calculations, and interactive dashboard development.
Microsoft Excel / CSV Files – Raw data storage and preprocessing.
Power Query – Data cleaning, transformation, and shaping.
DAX (Data Analysis Expressions) – KPI calculations and business metrics.
